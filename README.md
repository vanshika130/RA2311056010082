# CampusNotify — Frontend + REST API Contract

Real-time campus notification platform for placement drives, exam results, events, and announcements.

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

**Demo login:** `student@campus.edu` / `pass1234`
